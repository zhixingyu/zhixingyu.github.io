OpenCV学习笔记_1

Posted on 2019-01-15

加载、修改、保存图像

  • 加载图像cv::imread
  • 修改图像cv::cvtColor
  • 保存图像cv::imwrite

  1. 加载图像cv::imread

    • imread功能是加载图像文件成为一个Mat对象，其中第一个参数表示图像文件名称。
    • 第二个参数表示加载图像的类型，支持常见的三个参数值。
    • IMREAD_UNCHANGED(<0)表示加载原图，不作任何改变。
    • IMREAD_GRAYSCALE(0)表示把原图作为灰度图像加载进来。
    • IMREAD_COLOR(>0)表示把原图作为RGB图像加载进来。
    • openCV支持JPG、PNG、TIFF等常见格式图像文件加载。

  2. 显示图像cv::namedWindow与cv::imshow

    • namedWindow功能时创建一个OpenCV窗口，由OpenCV自动获取与释放，无需手动销毁。
    • 常用：namedWindow("Window Title",WINDOW_AUTOSIZE)。
    • WINDOW_AUTOSIZE自动根据图像大小显示窗口大小，不能认为改变窗口大小。
    • WINDOW_NORMAL跟QT集成时，允许修改窗口大小。
    • imshow根据窗口名称显示图像到指定窗口上去，第一个参数是窗口名称，第二个参数是Mat对象。

  3. 修改图像cv::cvtColor

    • cvtColor是把图像从一个色彩空间转换到另一个色彩空间，三个参数：源图像，转换之后的图像，源和目标色彩空间（COLOR_BGR2HLS、COLOR_BGR2GRAY）
    • cvtColor(image,gray_image,COLOR_BGR2GRAY)

  4. 保存图像cv::imwrite

    • 保存图像文件到指定目录。
    • 只有8位，16位的PNG、JPG、Tiff文件格式而且是单通道或者三通道的BGR的图像才可以通过这种方式保存。
    • 保存JGP格式的时候可以保存透明通道的图片。
    • 可以指定压缩参数。

  5. 代码

    #include<opencv2/opencv.hpp>
#include<iostream>
using namespace cv;
int main()
{
    Mat src = imread("D:/Documents/Pictures/test1.jpg");
    if (src.empty()) {
        printf("could not open\n");
        return -1;
    }
namedWindow("test opencv set up",CV_WINDOW_AUTOSIZE);
imshow("test opencv set up",src);

namedWindow("output windows",CV_WINDOW_AUTOSIZE);
Mat output_image;
cvtColor(src, output_image,CV_BGR2HLS);
imshow("output windows",output_image);

imwrite("D:/Documents/Pictures/test2.jpg",output_image);
waitKey(0);
return 0;
}

矩阵的掩膜操作

  • 获取图像像素指针
  • 掩膜操作

  1. 获取图像像素指针

    • CV_Assert(myImage.depth()==CV_8U);
    • Mat.ptr(int i=0)获取像素矩阵的指针，索引i表示行数，从0行开始计数。
    • 获得当前行指针 const uchar* current=myImage.ptr<uchar>(row);
    • 获取当前像素点p(row,col)的像素值 p(row,col)=current[col]

  2. 像素范围处理saturate_cast

    • saturate_cast<uchar>(-100)返回0
    • saturate_cast<uchar>(288)返回255
    • saturate_cast<uchar>(100)返回0
    • 这个函数的功能时确保RGB值的范围在0~255之间

  3. 掩膜操作

  4. 调用filter2D功能

    • 定义掩膜：
      Mat kernel=(Mat_<char>(3, 3) << 0, -1, 0, -1, 5, -1, 0, -1, 0);
    • filter2D(src,dst,src.depth(),kernel);

  • 使用filter2D
#include<opencv2/opencv.hpp>
#include<iostream>
#include<math.h>

using namespace cv;

int main(int argc, char ** argv) {
    Mat src, dst;
    src = imread("D:/documents/pictures/test1.jpg");
    if (!src.data) {
        printf("could not open image\n");
        return -1;
    }
    namedWindow("input image", CV_WINDOW_AUTOSIZE);
    imshow("input image",src);

    double t = getTickCount();
    Mat kernel = (Mat_<char>(3, 3) << 0, -1, 0, -1, 5, -1, 0, -1, 0);//定义掩膜
    filter2D(src,dst,src.depth(),kernel);
    double cost = (getTickCount() - t) / getTickFrequency();
    printf("time spend:%.2f",cost);
    namedWindow("contrast image demo", CV_WINDOW_AUTOSIZE);
    imshow("contrast image demo",dst);

    waitKey(0);
    return 0;
}
