单变量线性回归

代价函数

  • 其中m是样本数量，h(x)是预测函数，θ是模型参数
  • 我们的目的是要选择这两个参数值
  • 在线性回归中，我们有一个训练集，我们要做的是得出这两个参数值让预测函数表示的直线尽量与数据点很好地拟合。
  • 也就是说要选择使得h(x)最接近于训练样本中的y值，此时的θ值。

  • 可以列出代价函数的公式，代价函数也被称作平方误差函数，有时也被称为平方误差代价函数

  • 我们要做的就是关于θ0和θ1对函数J(θ0,θ1)求最小值。

代价函数(一)

  • 我们使用一个简化的假设函数更好地可视化J，如下图右边的简化函数

  • 如果使θ0=0，那么h(x)就是一条过（0,0）点的直线

  • hθ(x)是关于x的函数，而J(θ)是关于θ的函数，θ代表直线的斜率,假设训练集如下

    • 假设θ1=1，此时hθ(x)=x，那么求得J(1)=0，也就是说J(θ)过（1,0）点
    • 假设θ1=0.5，此h(x)的斜率是0.5，那么求得J(0.5)≈0.58
    • 假设θ1=0，此h(x)=0，那么求得J(0)≈2.3
    • 以此类推，可以绘制出J(θ1)的图像
    • 其中每一个θ1的值对应着hθ(x)的一条直线，当θ1=1时，J是最小值为0，事实上此时是最好的符合数据的直线

梯度下降法 (Gradient descent)

  • 假设从函数表面任一点出发寻找局部最优解，把这个图像想象成一座山，想尽快走下山，那么应该朝向哪里迈步，最佳的下降方向如图所示，直至收敛至局部最低点

  • 梯度下降法的特点是假如一开始出发的点向右偏移了一点，那么使用梯度下降法就会寻找到另外一个局部最低点

  • 梯度下降法定义如下
    • α是学习率，决定下山时迈多大的步子，α越大步子越大
    • := 是赋值运算符

梯度下降知识总结

  • 如果α值很小，那么梯度下降法的速度会很慢，因为步子很小，要一点一点挪动
  • 如果α值很大，会移动一大步，有可能会错过局部最低点，会导致无法收敛甚至发散
  • 假设初始值已经在局部最低点，那么这点的切线斜率为0，因此导数项为0，那么θ值就不变
  • 在梯度下降法中，当我们接近局部最低点时， 梯度下降法会自动采取更小的幅度，是因为当接近局部最低点时，根据定义，在局部最低处导数为0， 所以当接近局部最低时导数值会自动变得越来越小，所以梯度下降会自动采取较小的幅度，实际上并没有必要再另外减小α 。

线性回归的梯度下降

  • 将梯度下降法运用于最小化平方差代价函数

  • Batch梯度下降：每一步梯度下降都遍历整个训练集的样本， 在梯度下降中当计算偏导数时我们计算总和。
