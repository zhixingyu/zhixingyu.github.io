机器学习课堂笔记

绪论

什么是机器学习？

  • Arthur Samuel:Feild of study that gives computers the alibity to learn without being explicitly programmmed.
  • Tom Mitchell:A computer program is said to learn from experience E(经验E) with respect to some task T(任务T) and some performance measure P(性能度量P),if its performance on T,as measured by P,improves with experience.
    • T:给邮件分类
    • E:观察你对邮件的标签（垃圾邮件或者非垃圾邮件）
    • P:正确归类的邮件比例

分类

  • Supervised learning(监督学习)
  • Unsupervised learning(无监督学习)
  • Others:Reinforcement learning(强化学习),recommender systems(推荐系统)

Supervised learning(监督学习)

  • 监督学习：给算法一个数据集，其中包含了正确答案(right answers)，算法的目的是给出更多的正确答案。

    • 例如已知一些房价来预估给定房价，由于已知的房价是正确的数值,我们设法预测连续值得到预估值。

  • 回归问题（regression problem）: 目的是预测一个连续值输出。

  • 分类问题（classification problem）：目的是预测离散值输出。

Unsupervised learning(无监督学习)

  • 聚类算法:将未知的数据自动按得到的类型分成簇
    • 把这些数据分成两个不同的簇(clusters)

  • 应用：组织大型计算机群，社交网络分析，市场分割，天文数据分析。
  • 鸡尾酒会问题：假设一个鸡尾酒会上只有两个人，两人同时说话，两个麦克风会录到两人的混合声音，通过编程处理音频，将两人的声音分离。
