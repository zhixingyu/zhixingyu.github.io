开操作

  • 先腐蚀后膨胀
  • 可以去掉小对象，假设对象是前景色，背景是黑色。
  • API
    • morphologyEx(src, dst, CV_MOP_BLACKHAT, kernel);
      • Mat src:输入图像
      • Mat dst:输出结果
      • int OPT： 形态学操作类型CV_MOP_BLACKHAT/CV_MOP_OPEN/CV_MOP_CLOSE/CV_MOP_TOPHAT
      • Mat kernel:结构元素
      • int Iteration:迭代次数，默认是1
  • 代码

闭操作

  • 先膨胀后腐蚀
  • 可以填充小的洞，假设对象是前景色，背景是黑色。

形态学梯度

  • 膨胀减去腐蚀
  • 又称为基本梯度（其他还包括：内部梯度、方向梯度）
